How to Use

Web Speech API Mode (Recommended)

  1. Ensure you're using Chrome or Edge browser
  2. Grant microphone permission when prompted
  3. Tap START to begin recording
  4. Speak naturally - transcription appears in real-time
  5. The app will keep listening even during pauses
  6. Tap STOP when finished
  7. Use COPY or EXPORT to save your transcript
