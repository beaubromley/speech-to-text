How to Use

  1. Use Chrome or Edge browser
  2. Allow microphone access
  3. Click START and speak normally
  4. Text appears in real-time
  5. Click STOP when done
  6. Copy or export your transcript

Microphone Permissions

  • iOS: Settings → Safari → Microphone
  • Android: Chrome → Site Settings → Microphone
  • Desktop: Click lock icon in address bar

Notes

  • Needs HTTPS connection
  • Requires internet connection
  • Transcript auto-saves to browser
  • Works best in quiet environments
